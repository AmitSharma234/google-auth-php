# Google Authentication in PHP (OAuth 2.0)

A reusable Google Sign-In module using PHP & MySQL.

## Tech Stack
- PHP 8+
- MySQL
- Google OAuth 2.0
- Google PHP Client Library

## Features
- Secure Google login
- Automatic user registration
- Session-based authentication
- Clean modular architecture
